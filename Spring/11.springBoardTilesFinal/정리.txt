
 * 회원관리 및 자료실형태의 게시판

  1)회원관리 : 로그인 , 로그아웃


  2) 자료실 : 등록, 검색 , 읽기, 삭제, 수정
              (업로드 , 다운로드기능)
             *전제조건 : 자료실 이용은 반드시
                         로그인 된 사용자만 접근 가능.


  * 주요 적용기술
    - 화면 layout : tiles적용
    - 세션유무체크 : AOP적용
    - 업로드 , 다운로드 : 업로드 컴포포넌트
    - DB연동 : MyBatis 적용
    - transaction적용
    - 예외사항 : 별도의 예외페이지를 설정하여 예외처리하도록.
    - UI : bootstrap 이용!


  * 필요한 lib
    - sprint WEB MVC
    - tiles 3개
    - aop 1개
    - 업로드 관련 2개
    - DB+MyBatis (오라클, mybatis, log4 , spring-mybatis, jdbc, dbcp)
    - transction적용 : aop

----------------------------------------------
  *springBean 설정문서
    1) servlet-context.xml : 기본

    3) upDown-context.xml : 업로드 설정
